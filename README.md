## Prerequisites :open_file_folder:

* Python v3+

## Installation :hammer:

Install Virtual Environment :
```
$ pip install pipenv
```

Set env version :
```
$ pipenv --python 3.9
```

Activate env :
```
$ pipenv shell 
```

Install dependencies :
```
$ pipenv install -r requirements.txt
```

## Usage :stars:

To run the server on localhost.

```
$ python manage.py runserver
```
